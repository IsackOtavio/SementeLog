# Projeto de Gestão de Estoque

Este é um projeto de um sistema de gestão de estoque desenvolvido em Python, com uma interface web em HTML, CSS e JavaScript.

## Estrutura de Arquivos

O projeto foi reorganizado para uma melhor estrutura, seguindo as convenções de projetos web com Python:

- `app.py`: O arquivo principal da aplicação, responsável pela lógica do backend (provavelmente usando Flask ou outro framework).
- `templates/`: Contém todos os arquivos HTML que compõem a interface do usuário.
- `static/`: Contém os arquivos estáticos, como CSS e JavaScript, que são utilizados pelos arquivos HTML.
- `README.md`: Este arquivo, com a documentação do projeto.

## Como Executar o Projeto

Para executar o projeto, você precisa ter o Python instalado.

1. Instale as dependências (geralmente em um arquivo `requirements.txt`, que não foi encontrado mas pode ser necessário criar):
   ```bash
   pip install -r requirements.txt
   ```

2. Execute a aplicação:
   ```bash
   python app.py
   ```

A aplicação estará disponível em um endereço local, como `http://127.0.0.1:5000`.

## Próximos Passos

- Analisar o arquivo `app.py` para entender como as rotas e a lógica do backend estão implementadas.
- Verificar os arquivos HTML em `templates/` para garantir que os caminhos para os arquivos CSS e JavaScript em `static/` estão corretos.
- Criar um arquivo `requirements.txt` com as dependências do projeto.
